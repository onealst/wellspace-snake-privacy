1. Overview

This privacy policy explains how your data is handled when you use Wellspace Snake, a browser-based Chrome Extension developed to provide a fun and interactive experience.

2. What Data We Collect

Currently, Wellspace Snake does not collect or transmit any personal data.
The extension runs entirely in your browser and does not send any information to our servers.

3. Future Updates

In future versions, we may introduce optional features that allow users to submit game results to appear on a public billboard or leaderboard. If this feature is implemented:

  • Only non-personal, anonymized game data (e.g., score, timestamp, generic nickname) will be submitted.
  • Participation in the leaderboard feature will be completely optional.
  • This privacy policy will be updated accordingly before any such data collection begins.

4. Data Sharing

We do not share, sell, or rent your data to any third parties.
If any data collection is added in the future, it will be for the sole purpose of improving the user experience and will remain anonymized.

5. Security

Since no user data is currently collected or stored, there is no sensitive data at risk. Any future data-handling features will be implemented with industry-standard security practices.

6. Changes to This Policy

We may update this Privacy Policy to reflect new features or regulatory requirements. Updates will be posted on this page with an updated Effective Date, and any significant changes will be communicated through the Chrome Web Store or within the extension interface.
